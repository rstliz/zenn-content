# インターフェース

## 基本

- Goのインターフェースは明示的でははない、暗黙的である
- Goではインターフェースを実装する為のキーワードがない

### インターフェースの定義

例
```go
type Person interface {
	FullName() string
	Speak()  string
}
```
- インターフェースには実装の詳細を含まない
- インターフェースを実装する型にはインターフェースの全てのメソッドが必要となる
